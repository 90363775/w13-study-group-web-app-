Study Notes

This is an example of study notes in HTML. You can customize and add your own content.

  • Topic 1: HTML
    • HTML stands for HyperText Markup Language.
    • It is used to structure content on the web.
    • HTML elements are represented by tags.
  • Topic 2: CSS
    • CSS stands for Cascading Style Sheets.
    • It is used for styling HTML elements.
    • Selectors are used to target specific HTML elements for styling.
  • Topic 3: JavaScript
    • JavaScript is a scripting language used for web development.
    • It adds interactivity to web pages.
    • Commonly used for client-side scripting.
